## Egocentric Motion Labeling Project (Takara)

**Context:** Takara is working on a "goal reaching" motion research project. Needs to label egocentric motion data showing people walking toward target objects.

**Dataset:**
- Location: S3 `far-research-internal/jiamanli/datasets/for_vlk_motion_gen_data_rendered/sage_batch_walking/`
- Downloaded to: `/home/ubuntu/datasets/sage_batch_walking/`
- 1024 samples, ~3.2GB total
- Each sample has: `head_left.mp4`, `head_right.mp4` (stereo egocentric), `third_person.mp4`, `motion.npz`
- 4 seconds per video, 120 frames @ 30fps
- Folder names encode target object (e.g., `0001_839920_seq_000_cup`)
- High-quality rendered 3D scenes (kitchens, cafés, etc.)

**Model:**
- Initially suggested wrong model (`Jeremmmyyyyy/Qwen-2.5-QA` - text-only, gated)

- Running on 8x L40S GPUs (~45GB VRAM each)

**Labeling Pipeline:**
- Script: `/home/ubuntu/datasets/label_egocentric_motion.py`
- Extracts first + last frames from `head_left.mp4`
- Prompts model to identify goal object, location, scene description, confidence
- Output: `/home/ubuntu/datasets/labels.jsonl` (JSONL format)
- ~12 seconds per sample, estimated ~3.5 hours total
- Supports `--resume` flag to continue interrupted runs

**Status at 01:30 UTC:** Labeling job running (PID 1419930), 7+ samples completed so far.

**Notes:**
